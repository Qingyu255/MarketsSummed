About This Website

This website was created as a personal project to assist users in getting a quick overview of the US stock market conditions. This is done by helping to identify short term bullish or bearish divergences in the stock market indicators through various means, such as the use of contrarian indicators.

This website's frontend is constructed using HTML/CSS and bootstrap whilst the backend is built on Python (Object-Oriented Programming) and the Flask framework. The charts were created using the Plotly library and the chart studio API.
